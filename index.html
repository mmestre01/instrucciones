Instrucciones de Decodificación

Este sistema de codificación oculta un mensaje utilizando tres símbolos: ;, : y ?, que indican qué letra tomar de la frase que sigue a dicho símbolo.

Reglas de extracción:

  • ; → Toma la ultima letra del la palabra que tiene el símbolo.
  • : → Toma la siguiente letra del la palabra que tiene el símbolo.
  • ? → Toma la primeraletra de la frase con el símbolo.

Las comas (,) y otros signos no afectan al sistema de codificación.

Para decodificar el mensaje oculto, lee el texto completo y extrae las letras según las reglas anteriores, en orden de aparición de los símbolos.

Ejemplo:

Texto: "Lo hice; Aunque dudaba: ¿No estaba seguro? Pero; confié."
→ : → "Aunque dudaba: No" → n
→ ; → "Lo hice; Aunque dudaba" → e
→ ? → "¿No estaba seguro?" → n
Mensaje oculto: "nen"
